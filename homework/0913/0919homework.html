0913 linux shell命令学习

命令：

  • 列文件夹命令:ls
    ls -R 递归选项。列出了当前目录下包含的子目录中的文件
    1.转到20213609liuzx文件夹 -cd 20213609liuzx
    2.列出了当前目录下包含的子目录中的所有文件 ls -R
    ls -F 区分文件和目录，目录名称后面带了(/) ,会在可执行文件后面带上（*）,类似.publish.sh的文件。
    ls -a 显示所有文件目录，包括隐藏。
  • 重命名文件（移动）
    在Linux中，重命名文件称为移动，mv命令可以将文件和目录移动到另外一个位置或重新命名。
    例如 mv test_onea test_oneb //重命名
    tab键自动补全命令，非常实用
  • 解包命令:tar -xf mp3
    解析解包命令:tar -[主选项+辅选项] 文件或者目录,主选项，必须要有，辅选项是辅助使用的，可以选用。
    x:从文件包中释放文件，相当于解析包
    f:指定压缩包名称
  • 删包命令:rm LinuxCMD-1.mp3
    可能会出现的选项:rm: remove write-protected regular file 'LinuxCMD-1.mp3'?————这是删除保护文件夹的询问选项
    输入y即可
  • 用浏览器播放视频文件命令:
    #!/bin/sh
    "你的浏览器地址\chrome.exe" '你要打开的文件的地址'$1
    例如:
    #!/bin/sh
    $ "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe"'E:\Editor pr\0417/0417.MP4'$1
